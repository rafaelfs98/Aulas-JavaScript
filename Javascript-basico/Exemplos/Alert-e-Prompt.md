# Alert e Prompt 

### Exemplos
* exemplo 1 (alert)
``` js
alert("Teste")
var nome = prompt("qual e seu nome?");

alert("Boa tarde, " +nome);
```
* Exemplo 2 (prompt)

```js
var numero = parseInt(prompt("Digite um numero"));

var dobro = numero + numero;

alert("O dobro de " + numero + "e " + dobro);
